# Project: mi_primer_project_GSD

## Goal
Establish a clean, empty project structure using Java for the backend and React for the frontend to explore and test the GSD (Get Shit Done) framework capabilities.

## Tech Stack
- **Backend:** Java (Spring Boot)
- **Frontend:** React (TypeScript)
- **Architecture:** Decoupled Client-Server

## Success Criteria
- [ ] Functional Java backend skeleton.
- [ ] Functional React frontend skeleton.
- [ ] Basic communication between frontend and backend.
- [ ] GSD workflow files (.planning) fully initialized.
